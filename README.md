# miCoach-Hackathon-2026

### What is the miCoach Hackathon?

Our goal is to revolutionize the Product Discovery phase of the 'Digital Coach' for LATAM engineers, using the power of Artificial Intelligence to boost our region's talent.

#### How to submit your project

Each team will have its own folder in the main branch. Please make sure to upload your project **ONLY** to your specified folder before March 20th, 08:00pm PST. Include a readme file with the link to your team's repository for additional reference.
